О проекте

Данный проект представляет собой систему управления рекрутментом, разработанную в рамках курсового проекта. Система позволяет эффективно управлять процессом найма, включая работу с вакансиями, кандидатами и заявками на трудоустройство.

Использованные технологии
Java Spring Boot
PostgreSQL
Bootstrap 5
Thymeleaf
